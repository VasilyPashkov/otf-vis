# otf-vis
Visualizer for Open Trace Format (OTF) files.

## Description

## Key Features
* Open Trace Format (OTF) files visualization
* XML trace files visualization
* Qt and C++
* IDE Qt Creator

## Building

```
# Build prerequirements
$ sudo apt-get upgrade
$ sudo apt-get install
$ sudo apt-get install
```

To build otf-vis you must:
```
qmake
make
make install
```
